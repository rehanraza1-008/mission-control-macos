# Mission Control – macOS Productivity Automation System

Mission Control is a personal productivity automation system built for macOS.

It provides:

- Voice reminders for study sessions
- Prayer time announcements
- Automatic schedule tracking
- Live Mission Control dashboard
- Daily review reminders
- Time-block based productivity management

## Features

### Dashboard
A live HTML dashboard displaying:

- Current time
- Current task
- Upcoming task

### Voice Alerts

The system announces:

- Fajr wake-up alarms
- Study block transitions
- Prayer times
- Meal breaks
- Daily review sessions
- Sleep release

### Automation

Built using:

- Shell Scripting (Bash)
- HTML
- JavaScript
- macOS LaunchAgents
- Hammerspoon (Lua)

## Project Goal

To create a distraction-free personal command center that keeps users accountable to a strict daily schedule without relying on paid productivity software.
